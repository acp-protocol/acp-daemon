# Contributing

Thank you for your interest in contributing to ACP Daemon!

For contribution guidelines, please see the main guide:

**[ACP Contributing Guide](https://github.com/acp-protocol/acp-spec/blob/main/CONTRIBUTING.md)**

## Quick Start for Daemon Development

```bash
git clone https://github.com/acp-protocol/acp-daemon.git
cd acp-daemon
cargo build
cargo test
```

## Before Submitting

- [ ] `cargo fmt`
- [ ] `cargo clippy -- -D warnings`
- [ ] `cargo test`
